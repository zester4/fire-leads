# FireLeads: Advanced Lead Extraction System

**FireLeads** is a production-ready, highly efficient lead extraction system powered by **Firecrawl v2**. Designed for robust and validated lead generation, this system leverages the latest web scraping technologies and intelligent processing to deliver high-quality leads.

## Key Features

*   **Firecrawl v2 SDK Integration**: Utilizes the latest Firecrawl v2 API for superior web scraping capabilities, ensuring better reliability and performance.
*   **Batch Scraping**: Significantly faster lead processing through concurrent batch scraping, optimizing data collection workflows.
*   **Enhanced Email Validation**: Integrates the `email-validator` library with comprehensive DNS checks to ensure the highest accuracy and deliverability of extracted email addresses.
*   **Dual Output Formats**: Provides extracted leads in both `.txt` (plain text) and `.csv` (structured data) formats, offering flexibility for various downstream applications.
*   **Robust Error Handling**: Implements sophisticated retry logic and fallbacks to gracefully handle network issues, website changes, and other scraping challenges, ensuring continuous operation.
*   **Real-time Progress Tracking**: Features real-time progress bars to monitor the status of lead extraction jobs, providing transparency and control over long-running processes.
*   **Intelligent Filtering**: Employs smarter page candidate detection algorithms to focus on relevant content, minimizing noise and maximizing the quality of extracted information.

## Technologies Used

*   **Firecrawl v2 SDK**: Core web scraping engine.
*   **Python-dotenv**: For managing environment variables securely.
*   **Email-validator**: For robust email format and DNS validation.
*   **Dnspython**: Dependency for advanced DNS checks in email validation.
*   **Tqdm**: For displaying intelligent progress bars.
*   **Requests**: For making HTTP requests.
